Mis trabajos

Smartwatches Elite Store

Este proyecto es una landing page de un negocio de producto tecnológico con fines publicitarios, Smartwatches Elite Store donde resalta el logo representativo del negocio, los principales productos describiendo los modelos, funciones más resaltante y precios en promoción. Este trabajo fue elaborado con HTML en la estructura y CSS puro para la apariencia visual.

Iguanas Page

Este proyecto es una landing page con fines informativos referente a las iguanas. La importancia, habitat, principales características de este reptil en el ecosistema con imagenes ilustrativas e invitación para visitar un sitio web con mayor información. Proyecto maquetado y estilizado con CSS, utilizando Flexbox eficazmente en párrafos y tarjetas distribuidas.

Cuppon Page

El presente proyecto es una landing page de un negocio denominado CUPPON que ofrece promociones para los consumidores en los sectores turismo, gastronómico, entretenimiento, retail, entre otros. Este trabajo fue elaborado con Bootstrap para la maquetación y apariencia, personalizandolo en función al negocio y sus fines hacia su público objetivo.
